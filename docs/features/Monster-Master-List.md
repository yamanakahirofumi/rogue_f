# Monster-Master-List

## 1. 概要
本ドキュメントでは、RogueF における全モンスターのマスターリストを定義します。各モンスターの基本ステータス、AIパターン、獲得経験値、および捕獲難易度を規定し、実装およびバランス調整の基準とします。

## 2. モンスターパラメータの定義
各モンスターは以下の基本パラメータを持ちます。

- **HP**: 生命力。
- **攻撃力 (Attack)**: 与えるダメージに影響。
- **防御力 (Defense)**: 受けるダメージを軽減。
- **敏捷性 (Agility)**: 攻撃インターバルに影響。
- **器用さ (Dexterity)**: 命中率とクリティカル率に影響。
- **素早さ (Speed)**: 回避率に影響。
- **運 (Luck)**: 各種確率判定に影響。
- **攻撃インターバル**: 攻撃を行ってから次の攻撃が可能になるまでの時間（秒）。

## 7. 捕獲難易度 (Capture Difficulty)
[モンスターシステム](Monster-System.md) における基礎捕獲率は通常 30% ですが、一部のモンスターには補正がかかります。

| カテゴリ | 対象 | 基礎捕獲率 |
| :--- | :--- | :--- |
| 標準 | 上記以外の全モンスター | 30% |
| 低 | 上級モンスター (Tier 3) | 10% |
| 極低 | 特殊モンスター (Special) | 1% |
