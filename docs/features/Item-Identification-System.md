# Item-Identification-System

## 1. 概要
アイテム識別システムは、プレイヤーがダンジョン内で入手したアイテムの真の正体や付加状態（呪い・祝福）を判明させるための仕組みです。未識別のアイテムを扱う際のリスクと、正体が判明した際の喜びを提供し、探索の戦略性を高めます。

## 2. アイテムの状態
全てのアイテムは、以下の「識別状態」と「付加状態」の組み合わせを持ちます。

### 2.1 識別状態
- **未識別 (Unidentified)**: アイテムの本来の名称や効果が隠されている状態。
  - 表示例: 「謎の薬」、「青い巻物」、「鉄の剣（未識別）」
- **識別済み (Identified)**: アイテムの真の名称と効果が判明している状態。

### 2.2 付加状態 (Status)
アイテム（主に装備品や一部の道具）には、以下の付加状態が存在する場合があります。
- **通常 (Normal)**: 特殊な補正がない標準的な状態。
- **祝福 (Blessed)**: アイテムの効果が強化されている、または特殊な恩恵がある状態。
- **呪い (Cursed)**: 装備すると外せなくなる、または負の効果が発生する状態。

## 3. 識別・鑑定の方法
未識別のアイテムを識別するには、以下の方法があります。

### 3.1 アイテムの使用 (Use)
- 薬や巻物などの消耗品は、実際に使用（飲む、読む等）することでその効果が発揮され、同時に識別済み状態になります。詳細は **[アイテムマスターリスト](Item-Master-List.md)** の「識別方法」を参照してください。
- ただし、マイナスの効果を持つアイテムを識別目的で使用するリスクが伴います。

### 3.2 装備 (Equip)
- 武器や防具は、装備することでその性能や呪いの有無が判明します。
- 呪われていた場合、特定の手段（解呪の巻物等）を用いない限り、その装備を外すことができなくなります。

### 3.3 鑑定用アイテムの使用
- 「鑑定の巻物」などの専用アイテムを使用することで、インベントリ内の未識別アイテムを安全に識別できます。

### 3.4 店での鑑定
- ダンジョン内のショップにおいて、店主に有料で鑑定を依頼できます。
- **鑑定料**: アイテムの基本価格（[アイテムマスターリスト](Item-Master-List.md) の `value`）の **20%**。
  - 基本価格（Base Price）に基づき算出され、市場価格（Market Price）の変動の影響を受けません。
  - ただし、最低鑑定料を **50 ゴールド** とします。

## 4. 呪いと祝福のメカニズム

### 4.1 呪いの影響
- **装備の固定**: 呪われた武器や防具を装備すると、通常の操作では変更・解除ができなくなります。
- **性能低下**: 攻撃力や防御力にマイナス補正（一律 **-1**）がかかります。
- **特殊効果の阻害**: 本来の効果が発動しなくなる場合があります。

### 4.2 祝福の恩恵
- **性能向上**: 攻撃力や防御力にプラス補正（一律 **+1**）がかかります。
- **消費の抑制**: 使用しても一定確率で消費されない、または効果時間が延長される場合があります。

## 5. データモデルとの関連
本システムの状態は、`InventoryItem` インターフェースの以下のプロパティで管理されます。

- `isIdentified`: 識別状態（true=識別済み, false=未識別）
- `isCursed`: 呪い状態（true=呪い）
- `isBlessed`: 祝福状態（true=祝福）
