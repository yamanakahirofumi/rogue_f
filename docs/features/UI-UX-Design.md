# UI-UX-Design

## 1. 画面構成とルーティング
アプリケーションは以下のルートで構成されています。

- `/user/create`: **ユーザー作成画面**
  - 新規プレイヤーの作成を行います。
- `/play`: **プレイ画面**
  - ダンジョン探索のメイン画面です。
- `/admin`: **管理画面**
  - 世界設定、アイテム設定、メニュー管理などを行います。
  - `/admin/menu`: 管理メニューを表示します。
  - `/admin/menu/menu`: 世界設定（`WorldComponent`）を表示します（子ルート）。

## 2. コンポーネント階層
主なコンポーネントの階層構造は以下の通りです。

- `AppComponent`
  - `CreateUserComponent` (ユーザー作成)
  - `DungeonComponent` (プレイ画面)
    - `MessageComponent` (メッセージ表示)
    - `StatusBarComponent` (ステータス表示)
  - `AdminModule` (遅延読み込み)
    - `MenuComponent` (管理メニュー)
      - `WorldComponent` (世界設定、子ルートとして表示)

## 3. 基本デザイン
- **背景色**: 黒 (`background: black`)
- **文字色**: 白 (`color: white`)
- **フォント**: `"Courier New"`, `"Noto Sans Mono CJK"`, monospace (ローグライクらしいレトロな雰囲気を演出)
- **操作フィードバック**:
  - ゲーム内メッセージ（`MessageComponent`）によるイベントログ出力。
  - ステータスバー（`StatusBarComponent`）によるリアルタイムな状態表示。
    - **HP/スタミナ**: プログレスバーで視覚的に表示。
    - **満腹度**: アイコンの色変化や数値で空腹状態を警告。
    - **状態異常**: 付与されているバフ・デバフのアイコンを表示し、ホバーで詳細を確認可能。

## 4. 管理者用 UI (Admin UI)
管理画面は、世界の構築と運営を効率的に行うための専用インターフェースを提供します。

### 4.1 メニュー構成
- **Dungeon タブ**:
    - ダンジョンの新規作成・編集。
    - リアルタイムマップ上でのモンスター・トラップ・施設のドラッグ＆ドロップ配置。
- **Item タブ**:
    - 倉庫内のアイテム一覧表示、ソート、フィルタリング。
    - ショップへの陳列アイテム選択と価格設定。
- **World タブ**:
    - 接続中の他サーバー一覧と信頼関係の状態表示。
    - 世界の基本ルール（PVPのON/OFF等）のトグルスイッチ。

### 4.2 操作フィードバック
- 管理アクション（配置、価格変更等）の成功・失敗をトースト通知やログで即座に通知。
- リアルタイムで変動する「基本市場価格」のグラフ表示。

## 5. ミニマップ機能 (Mini-map)
探索済みのエリアと現在位置を把握するための補助的な UI です。

### 5.1 表示仕様
- **配置**: プレイ画面（`/play`）の右上または右下にオーバーレイ表示。
- **サイズ**: ダンジョン全体の約 1/4 〜 1/6 程度の縮尺、または周囲 15x15 マス程度の固定サイズ（切り替え可能）。
- **透過度**: 背景が透けて見えるよう適度なアルファ値を設定。

### 5.2 描画要素
- **壁 (Wall)**: 灰色または暗い色で描画。
- **通路・部屋 (Floor)**: 明るい灰色または白で描画（探索済みの場所のみ）。
- **プレイヤー (Player)**: 黄色または白色の点（現在位置）。
- **階段 (Stairs)**: 下り階段は青、上り階段は緑のアイコンまたは点で表示（発見済みの場合のみ）。
- **ショップ (Shop)**: ドルのアイコンまたは特定の記号で表示（発見済みの場合のみ）。
- **扉 (Door)**: 茶色またはオレンジ色の点 (`+`) で表示。
- **回復の泉 (Recovery Spring)**: 水色の点 (`&`) で表示。
- **転送門 (Teleport Gate)**: 紫色の点 (`O`) で表示。
- **トラップ (Trap)**: 赤色の点 (`^`) で表示（発見済みの場合のみ）。

### 5.3 フォグ・オブ・ウォー (Fog of War)
- 未探索のエリアはミニマップ上では完全に黒く（非表示）なります。
- プレイヤーが移動することで、その周囲の視界が確保され、ミニマップが徐々に更新されます。
