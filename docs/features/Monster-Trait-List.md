# Monster-Trait-List

## 1. 概要
モンスター特性（Traits）は、モンスター個体が持つ特別な能力や性質です。特性はステータスの補正や、戦闘、探索における特殊な効果をもたらします。特性は野生個体が稀に持っているほか、[モンスター繁殖システム](Monster-Breeding-System.md) を通じて子に継承されることがあります。

## 2. 特性パラメータ
各特性は以下の属性を持ちます。

- **ID**: システム上の識別子。
- **名称**: 特性の表示名。
- **効果**: 特性がもたらす具体的なメリット（またはデメリット）。
- **レアリティ**: 特性の希少度（Common, Rare, Epic）。継承率や突然変異率に影響します。

## 3. ステータス補正系特性
基礎ステータスを強化または変化させる特性です。

| ID | 名称 | 効果 | レアリティ |
| :--- | :--- | :--- | :--- |
| `hard_skin` | 硬い皮膚 | 防御力 +15% | Common |
| `sharp_claws` | 鋭い爪 | 攻撃力 +10%, クリティカル率 +5% | Common |
| `quick_reflexes` | 素早い反応 | 素早さ +15%, 回避率 +5% | Common |
| `sturdy_body` | 頑丈な体 | 最大HP +20% | Common |
| `keen_eye` | 鋭い眼光 | 器用さ +15%, 命中率 +10% | Common |
| `lucky_clover` | 幸運の持ち主 | 運 (Luck) +5 | Rare |

## 4. 戦闘・特殊効果系特性
特定の状況下で発動する、またはステータス以外の要素に影響する特性です。

| ID | 名称 | 効果 | レアリティ |
| :--- | :--- | :--- | :--- |
| `poison_touch` | 毒の体 | 物理攻撃時に 15% の確率で対象を「毒」状態にする。 | Rare |
| `regenerative` | 自己再生 | 5秒ごとに最大HPの 1% を回復する。 | Epic |
| `mana_absorb` | 魔力吸収 | 攻撃時に与えたダメージの 5% 分、自身のスタミナ（または魔力）を回復する。 | Rare |
| `cowardly_dash` | 逃げ足 | HPが 25% 以下の時、移動速度が 1.5 倍になる。 | Common |
| `alpha_soul` | アルファの魂 | 同階層にいる味方モンスターの攻撃力を +5% する。 | Epic |
| `iron_will` | 鉄の意志 | スタン状態になる確率を 50% 軽減する。 | Rare |

## 5. 繁殖と継承
- 特性は繁殖時に親から子へ継承される可能性があります。詳細は [モンスター繁殖システム](Monster-Breeding-System.md) を参照してください。
- 突然変異により、両親が持っていない特性が発現することもあります。
