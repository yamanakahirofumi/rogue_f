# Item-Master-List

## 1. 概要
本ドキュメントでは、RogueF における全アイテムのマスターリストを定義します。各アイテムの具体的な効果、基本価格、および世界内存在上限（サーキュレーション制限）を規定し、実装の基準とします。

## 2. 食料 (Food)
[満腹度システム](Hunger-System.md) に基づき、プレイヤーの満腹度を回復させます。

| ID | 名称 | 回復量 | 追加効果 | 基本価格 | 上限 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `apple` | リンゴ | 20 | なし | 50 | なし |
| `ration` | 携帯食料 | 50 | なし | 150 | なし |
| `large_meat` | 大きな肉 | 100 | 最大満腹度 +1 | 500 | 100 |
| `rotten_food` | 腐った食料 | 30 | 30%の確率で「毒」状態 | 20 | なし |

## 3. 巻物 (Scrolls)
使用することで即座に特殊な効果を発動します。

| ID | 名称 | 効果 | 識別方法 | 基本価格 | 上限 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `scroll_identify` | 鑑定の巻物 | インベントリのアイテム1つを識別 | 使用 | 200 | なし |
| `scroll_uncurse` | 解呪の巻物 | 装備中の呪われたアイテムを外す | 使用 | 300 | なし |
| `scroll_light` | 明かりの巻物 | 階層内の全トラップ・マップを可視化 | 使用 | 500 | 50 |
| `scroll_teleport` | 高飛びの巻物 | 階層内のランダムな位置へ移動 | 使用 | 150 | なし |

| ID | 名称 | 効果 | 識別方法 | 基本価格 | 上限 |
| :--- | :--- | :--- | :--- | :--- | :--- |

| `potion_hp_large` | 回復の薬 | HPを全回復 | 飲む | 800 | 30 |
| `potion_antidote` | 毒消し草 | 「毒」状態を解除 | 飲む | 150 | なし |
| `potion_stamina` | スタミナの薬 | スタミナを 50 回復 | 飲む | 200 | なし |
