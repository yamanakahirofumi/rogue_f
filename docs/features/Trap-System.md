# Trap-System

## 1. 概要
トラップシステムは、ダンジョン内に設置された罠によってプレイヤーの探索を妨害し、緊張感と戦略性を与える仕組みです。管理者は自身のダンジョンにトラップを戦略的に配置することで、侵入者に対する防衛力を高めることができます。

## 2. トラップの種類
トラップはその効果に応じていくつかのカテゴリに分類されます。

### 2.1 ダメージ系トラップ
プレイヤーのHPを直接減少させます。
- **トゲの床 (Spikes)**: 踏むと固定ダメージを受ける。
- **地雷 (Landmine)**: 踏むと爆発し、周囲 1 マスを含めたダメージを与える。

### 2.2 状態異常系トラップ
プレイヤーに一時的な弱体化を与えます。詳細は [状態異常システム](Status-Effect-System.md) を参照してください。
- **毒矢 (Poison Needle)**: 踏むと「毒」状態になる。
- **警報 (Alarm)**: 踏むと大きな音が鳴り、周囲のモンスターがプレイヤーを追尾するようになる。また、一定確率で「スタン」状態を引き起こす。
- **鈍足の罠 (Slow Trap)**: 踏むと一定時間「鈍足」状態になる。

### 2.3 ユーティリティ系トラップ
プレイヤーの位置やリソースに干渉します。
- **ワープの罠 (Teleport Trap)**: 同じ階層のランダムな座標に転送される。
- **召喚の罠 (Summon Trap)**: 周囲にモンスターを数体召喚する。
- **装備外しの罠 (Equipment Remover)**: (検討中) 装備している武器や防具が強制的に外される。

## 3. 発見と解除

### 3.1 隠れた状態 (Hidden State)
- ほとんどのトラップは、設置された時点ではプレイヤーから見えない「隠れた状態」になっています。
- 隠れたトラップの上を歩くと、トラップが発動します。

### 3.2 探索 (Search)
- プレイヤーは周囲を「探索」するアクションを行うことで、隠れたトラップを発見できる可能性があります。
- **発見確率計算式**:
    - `発見確率 = (器用さ * 2 + 運) / (トラップの難易度 * 0.5)`
- 運（Luck）や特定のステータス、または「明かりの巻物」などのアイテムを使用することで、トラップを可視化できます。

### 3.3 解除 (Disarming)
- 発見されたトラップに対して、隣接した状態で「解除」を試みることができます。
- **解除成功率計算式**:
    - `解除成功率 = (器用さ * 3 + 運) / トラップの難易度`
- 解除に失敗すると、その場でトラップが発動するリスクがあります。

## 4. 管理者による配置

### 4.1 「マイ・ダンジョン」での運用
- 管理者は、自身が運営するダンジョンにおいて、所持している「トラップ資材」やゴールドを消費してトラップを配置できます。
- モンスターの配置（[モンスターシステム](Monster-System.md)）と組み合わせることで、より強力な防衛網を構築できます。
  - 例: 鈍足の罠の周囲に、遠距離攻撃を持つモンスターを配置する。

### 4.2 配置制限
- 1 つの部屋や階層に配置できるトラップの総数には、ダンジョンのレベルに応じた上限があります。
- 階段の直上など、プレイヤーが回避不能な場所への配置には制限がかかる場合があります。
