# PK-System

## 1. 概要
PK（プレイヤーキル）システムは、プレイヤー間の競争と緊張感を生み出すための要素です。本作では、単なる対戦にとどまらず、プレイヤーがモンスターとして介入するという独自の仕組みを採用しています。

## 2. 役割の流動性
全てのユーザーは、自身の世界を管理する**「管理者」**としての側面と、他の世界やランダムダンジョンを探索する**「プレイヤー」**としての側面の両方を持ちます。

## 3. モンスターとしての参戦
プレイヤーは特定の条件下で、**「モンスターの一種」**として他のプレイヤーのダンジョンに介入（乱入）することが可能です。
- **介入方法**: 特定のアイテムやコマンドを使用することで、他者の攻略中のダンジョンにモンスターとして出現します。
- **目的**: 探索中のプレイヤーを排除し、そのプレイヤーの所持品や経験値の一部を奪取することを目指します。
- これにより、従来のNPCモンスターにはない、高度な戦術的駆け引きが発生します。

## 4. PK時の報酬とペナルティ
- **PK報酬**: 撃破したプレイヤーが落としたアイテムやゴールドを獲得できます。
- **報酬の優先順位**:
    - **プレイヤー(PKer)による撃破**: 落とした全アイテム・ゴールドは、その場にドロップされ、撃破したPKerが優先的に取得権利を持ちます。
    - **環境・NPCによる撃破**: 落とした全アイテム・ゴールドは、[機能仕様書](Functional-Specification.md#5-1-管理者の利益（略奪と収集）) に基づき、管理者の倉庫へ直接転送されます。
- **PKペナルティ**: 撃退された場合、以下のデメリットが発生します。
    - **介入リソースの喪失**: 乱入に使用した特定のアイテム（乱入のオーブ等）を消費します。
    - **モンスターレベルの減少**: モンスター形態のレベルが 1 減少します。
    - **一定時間の再介入禁止**: 同一プレイヤーまたは同一サーバーへの一定時間（例: 30分）の再介入が制限されます。
