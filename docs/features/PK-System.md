# PK-System

## 1. 概要
PK（プレイヤーキル）システムは、プレイヤー間の競争と緊張感を生み出すための要素です。本作では、単なる対戦にとどまらず、プレイヤーがモンスターとして介入するという独自の仕組みを採用しています。

## 2. 役割の流動性
全てのユーザーは、自身の世界を管理する**「管理者」**としての側面と、他の世界やランダムダンジョンを探索する**「プレイヤー」**としての側面の両方を持ちます。

## 3. モンスターとしての参戦
プレイヤーは特定の条件下で、**「モンスターの一種」**として他のプレイヤーのダンジョンに介入（乱入）することが可能です。
- **介入方法**: 特定のアイテムやコマンドを使用することで、他者の攻略中のダンジョンにモンスターとして出現します。
- **目的**: 探索中のプレイヤーを排除し、そのプレイヤーの所持品や経験値の一部を奪取することを目指します。
- これにより、従来のNPCモンスターにはない、高度な戦術的駆け引きが発生します。

## 4. PK時の報酬とペナルティ
- **PK報酬**: 撃破したプレイヤーが落としたアイテムやゴールドを獲得できます（[デスペナルティ](Functional-Specification.md#4-デスペナルティの設定)の設定に依存）。
- **PKペナルティ**: 撃退された場合、モンスターとしてのステータス低下や、介入に使用したリソースの喪失が発生する可能性があります（検討中）。
