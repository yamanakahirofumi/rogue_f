# Monster-System

## 1. 概要
モンスターシステムは、管理者のダンジョンを構成する中心的な要素です。プレイヤーは野生のモンスターを捕獲し、自身のダンジョンに配置したり、繁殖させたりすることができます。

## 2. モンスターの獲得
モンスターは以下の方法で獲得できます。
- **捕獲**: ランダムダンジョン（フロンティア）に出現する野生のモンスターに対し、**「捕獲の球」**を使用して仲間にします。
    - 弱らせることで捕獲率が上昇します。
    - 捕獲したモンスターは「ストック（[倉庫システム](Warehouse-System.md)）」に送られ、管理者が自由に扱えるようになります。
- **孵化**: 拠点（管理者エリア）で「卵」を孵化させます。詳細は [モンスター繁殖システム](Monster-Breeding-System.md) を参照してください。

## 3. 繁殖と卵
- **卵の入手**: 所持しているモンスター同士を掛け合わせることで、新たな個体の卵を入手できます。
- これにより、強力な個体を増やしたり、ダンジョンの守りを固めたりすることが可能です。
- 詳細は [モンスター繁殖システム](Monster-Breeding-System.md) を参照してください。

## 4. モンスターの運用
- **配置**: 管理者は自身の「マイ・ダンジョン」に、所持しているモンスターを自由に配置できます。
- **育成**: モンスターは戦闘や特定のアイテムを通じて成長し、ステータスが向上する可能性があります（検討中）。
- **モンスターとしての介入**: プレイヤーは特定の条件下でモンスターとして他者のダンジョンに介入できます。詳細は [PKシステム](PK-System.md) を参照してください。

## 5. 捕獲アイテムの仕様

モンスターを仲間にするために必要な「捕獲の球 (Capture Ball)」の詳細です。

### 5.1 捕獲の球 (Capture Ball)
- **効果**: 野生のモンスターを捕獲し、倉庫へ転送する。
- **入手方法**:
    - **ショップ購入**: 500 ゴールド
- **捕獲成功率**:
    - `成功率(%) = 基礎捕獲率(30%) + ((最大HP - 現在HP) / 最大HP) * 60`
    - ※対象の HP が低いほど成功率が上がり、最大で 90% に達します。
