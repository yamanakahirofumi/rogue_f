# Construction-System

## 1. 概要
建築システムは、管理者が自身のダンジョン（マイ・ダンジョン）の構造を構築・カスタマイズするための仕組みです。探索で得た資材を消費して、独自の攻略ルートや防衛環境をデザインすることができます。

## 2. 建築資材 (Materials)
建築には、探索などで入手し、[倉庫システム](Warehouse-System.md)に保管される資材が必要です。
詳細は [管理者データモデル](../implementation/Admin-Data-Models.md#62-基本資材-basic-materials) の `StoredMaterial` を参照してください。

### 2.1 資材のカテゴリ
- **基本資材**: 壁、床、扉などの基本的な構造物の設置に使用します。
- **特殊資材**: トラップの強化、魔法的な仕掛け、特殊な施設の作成に使用します。
- **装飾資材**: ダンジョンの視覚的な演出（松明、彫像、タイルパターン）に使用します。

### 2.2 入手方法
- **探索 (Explore)**: ランダムダンジョン内での拾得。
- **解体 (Dismantle)**: 設置済みの構造物やトラップを撤去する際、一部の資材を回収できます。
- **購入 (Purchase)**: [ショップシステム](Shop-System.md)を通じて、他の管理者やシステムショップから購入します。

## 3. 建築・編集プロセス
管理者は管理画面（`/admin`）の「Dungeon」セクションから、以下の操作を行います。

### 3.1 マップエディタ
- **床の敷設**: プレイヤーが移動可能なエリアを定義します。
- **壁の配置**: 通行不能な境界を作成し、部屋の形状を決定します。
- **通路の接続**: 部屋間を繋ぐ通路を設計します。

### 3.2 施設の設置
特定の機能を持つ固定オブジェクトを配置できます。
詳細は [管理者データモデル](../implementation/Admin-Data-Models.md#61-施設種別-facility-types) の `PlacedFacility` を参照してください。

- **ショップカウンター (`shop_counter`)**: [ショップシステム](Shop-System.md)を稼働させるための施設。
- **回復の泉 (`recovery_spring`)**: プレイヤーのHPやスタミナを回復させる施設。
- **転送門 (`teleport_gate`)**: 階層内の特定の地点へのワープポイント。

## 4. 防衛とカスタマイズ
構築された地形に基づき、侵入者を迎撃するための要素を追加します。

### 4.1 モンスターとトラップの同期
- **配置**: [モンスターシステム](Monster-System.md)および[トラップシステム](Trap-System.md)に基づき、地形に合わせた戦略的な配置を行います。
- **地形効果**: 特定の地形（水、溶岩、砂地）がモンスターの能力やトラップの発動に影響を与える場合があります。

## 5. 建築の制限とコスト

### 5.1 配置容量 (Capacity)
1つのダンジョンまたは階層に配置できる構造物や施設の総数には、ダンジョンレベルに応じた上限があります。
- **最大容量の計算式**: `Capacity = 100 + (DungeonLevel * 50)`

配置物ごとの容量消費（目安）は以下の通りです。
| 配置タイプ | 容量消費 | 備考 |
| :--- | :--- | :--- |
| **モンスター** | 5 | [モンスターシステム](Monster-System.md) 参照。 |
| **トラップ** | 2 〜 8 | 種類により変動。詳細は [トラップシステム](Trap-System.md) 参照。 |
| **一般施設** | 10 〜 20 | 回復の泉、転送門など。 |
| **ショップ** | 10 | [ショップシステム](Shop-System.md) 参照。 |

※壁や床などの基本地形タイルは容量を消費しませんが、設置に資材が必要です。

### 5.2 建築・拡張コスト
構造物の設置やダンジョンの拡張には、資材とゴールドが必要です。

- **階層の追加 (Expand Floor)**: ダンジョンに新しい階層を追加します。
    - **コスト**: 10,000 ゴールド、魔力結晶 (`magic_crystal`) × 50
- **基本構造物**:
    - **壁 (Wall)**: 石材 (`stone`) × 2
    - **扉 (Door)**: 木材 (`wood`) × 5、鉄材 (`iron`) × 1
