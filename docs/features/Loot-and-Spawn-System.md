# Loot-and-Spawn-System

## 1. 概要
ドロップ品および出現システムは、ダンジョン探索における報酬の質と量を制御する仕組みです。ランダム生成ダンジョン（フロンティア）におけるアイテムの自然発生と、モンスターを撃破した際のドロップロジックを定義し、ゲームバランスを維持します。

## 2. フィールド出現ロジック (Floor Spawn)
ランダム生成ダンジョンの各階層が生成される際、以下のルールに基づいてアイテムおよびゴールドが配置されます。

### 2.1 出現密度
- **アイテム**: 1 階層につき `5 〜 12` 個。
- **ゴールド**: 1 階層につき `3 〜 8` 個。
- **階段・ショップ**: 1 階層につき各 1 つ（ショップは低確率出現）。

### 2.2 出現アイテムの選定
- 階層の深さ（Dungeon Level）に応じて、[アイテムマスターリスト](Item-Master-List.md) から出現可能なアイテムが抽選されます。
- 深層ほど上位の装備品や貴重な巻物が出現しやすくなります。

### 2.3 階層別出現テーブル
階層に応じた出現モンスターおよびアイテムの Tier 基準です。

| 階層範囲 | モンスター出現 | アイテム出現 | 備考 |
| :--- | :--- | :--- | :--- |
| **1F - 10F** | Tier 1 中心 | Tier 1 中心 | 序盤エリア。基本的な消耗品が主。 |
| **11F - 30F** | Tier 2 中心 (Tier 1 混在) | Tier 1 〜 2 | 中層への入り口。装備の重要性が増す。 |
| **31F - 60F** | Tier 2 中心 (Tier 3 稀に) | Tier 2 中心 (Tier 3 稀に) | 中層。強力なモンスターが出現し始める。 |
| **61F - 99F** | Tier 3 中心 | Tier 3 中心 | 深層。最高級の装備や資材が手に入る。 |

## 3. モンスタードロップロジック (Monster Drops)
モンスターがプレイヤーによって撃破された際、以下の確率でアイテムまたはゴールドをドロップします。

### 3.1 ドロップ確率
- **ゴールド**: 40%
- **アイテム**: 15% (モンスターの種類および「運」により補正あり)
- **何も落とさない**: 45%

#### 運 (Luck) によるドロップ補正
プレイヤーの「運」ステータスに応じて、アイテムおよびゴールドのドロップ率にボーナスが付与されます。
- `最終ドロップ率 = 基本ドロップ率 * (1 + (Luck / 100))`

### 3.2 モンスター階級別ドロップ傾向
- **Tier 1 (初級)**: 主に消耗品（リンゴ、弟切草）や少額のゴールド。
- **Tier 2 (中級)**: 装備品（鉄の剣、皮の鎧）や中額のゴールド。
- **Tier 3 (上級)**: 希少な巻物、上位装備、高額のゴールド。

## 4. サーキュレーション制限の影響
[機能仕様書](Functional-Specification.md) に基づき、世界内存在上限が設定されているアイテムについては、出現率が動的に変動します。

- **出現抑制**: あるアイテムの現在の流通数が上限に近い場合、そのアイテムの出現率が大幅に低下します。
- **出現停止**: 流通数が上限に達している場合、ランダムドロップおよびフィールド出現リストから一時的に除外されます。
- **優先補充**: 流通数が極めて少ない場合、出現率にボーナスが付与されることがあります。

## 5. 罠とエンティティの出現
フロンティアにおけるトラップおよびモンスターの初期配置ルールです。

### 5.1 トラップ出現
- **密度**: 1 部屋につき `1 〜 3` 個の隠れたトラップを配置。
- **種類**: 階層が深くなるほど、より難易度の高い（発見・解除しにくい）トラップが選定されます。

### 5.2 モンスター出現
- **初期配置**: 階層生成時に `8 〜 15` 体のモンスターがランダムな床に配置されます。
- **再出現 (Respawn)**: プレイヤーが一定ターン（時間）行動するごとに、未探索エリアに新しいモンスターが 1 体ずつ補充されます（上限あり）。

## 6. ゴールドの算出
ドロップおよび拾得時のゴールド額は以下の式で決定されます。

`獲得ゴールド = (基本額 * (1 + (現在の階層 / 20))) * 乱数(0.8 〜 1.2)`

- **基本額**:
  - フィールド拾得: 100
  - モンスタードロップ: 対象モンスターの `BaseExp * 2`
