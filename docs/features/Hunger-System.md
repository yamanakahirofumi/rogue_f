# Hunger-System

## 1. 概要
満腹度（サティエティ）システムは、プレイヤーの探索におけるリソース管理を深化させ、緊張感のあるゲーム体験を提供することを目的とします。プレイヤーは時間経過や行動によって減少する満腹度を、食料アイテムを摂取することで維持する必要があります。

## 2. 満腹度の基本メカニズム

### 2.1 パラメータ
- **満腹度 (satiety)**: プレイヤーの現在の空腹具合を表します。
- **最大満腹度 (maxSatiety)**: 満腹度の最大値（初期値は 100 を想定）。
これらの値は `Player` クラスにて管理されます。

### 2.2 満腹度の減少
満腹度は以下の条件で減少します。
- **時間経過**: 10秒ごとに 1 減少します。
- **行動消費**: プレイヤーが移動または攻撃を行うごとに一定値（例: 0.1）減少します。

## 3. 状態変化と影響
満腹度の値に応じて、プレイヤーに以下の影響を与えます。

| 状態 | 満腹度の範囲 | 影響 |
| :--- | :--- | :--- |
| **満腹 (Satiated)** | 80% - 100% | HPとスタミナの自然回復速度が 1.2 倍になる。 |
| **通常 (Normal)** | 20% - 79% | 通常の状態。 |
| **空腹 (Hungry)** | 1% - 19% | 自然回復（HP/スタミナ）が停止する。 |
| **飢餓 (Starving)** | 0% | 自然回復が停止し、5秒ごとに最大HPの 1% のダメージを受ける。 |

## 4. 食料アイテム
食料アイテムを使用することで満腹度を回復できます。

### 4.1 アイテム例
- **携帯食料 (Ration)**: 満腹度を 50 回復。
- **リンゴ (Apple)**: 満腹度を 20 回復。
- **大きな肉 (Large Meat)**: 満腹度を 100 回復し、最大満腹度が 1 上昇する。

### 4.2 特殊効果
- **腐った食料 (Rotten Food)**: 満腹度は回復するが、30% の確率で「毒」状態になる。

## 5. ゲームバランスの調整
- 満腹度の減少速度は階層が深くなるにつれて加速する可能性があります。
- スタミナ（短期リソース）と満腹度（長期リソース）を分けることで、プレイヤーに多層的なリソース管理を要求します。
