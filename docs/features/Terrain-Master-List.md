# Terrain-Master-List

## 1. 概要
本ドキュメントでは、RogueF における各種地形（Terrain）と、それがプレイヤーやモンスターに与える環境効果を定義します。地形はダンジョン生成時にランダムに配置されるほか、[建築システム](Construction-System.md) を通じて管理者が意図的に配置することが可能です。

## 2. 地形種別と効果一覧

各地形が持つ固有の効果と、描画に使用される記号の定義です。

| typeId | 名称 | 記号 | 環境効果 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| `water` | 水 | `~` | 行動インターバル (`actionInterval`) が **2.0 倍** になる。 | 水棲・飛行モンスターは影響を受けない。 |
| `lava` | 溶岩 | `!` | そのマスでの行動ごとに **5 HP** のダメージを受ける。 | 火属性・飛行モンスターは影響を受けない。 |
| `sand` | 砂地 | `:` | 行動インターバル (`actionInterval`) が **1.5 倍** になる。 | 砂漠適性・飛行モンスターは影響を受けない。 |

## 3. 属性による影響の相殺
特定の「特性（Traits）」や「属性」を持つエンティティは、地形のマイナス効果を無視、またはメリットに変換できる場合があります。

- **飛行 (Flying)**: 全ての地形トラップ・環境効果（ダメージ、減速）を無視する。
- **水棲 (Aquatic)**: 水地形での移動減速を無視し、逆に回避率が向上する。
- **火属性 (Fire Affinity)**: 溶岩によるダメージを無効化する。
