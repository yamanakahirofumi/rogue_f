# Monster-Breeding-System

## 1. 概要
モンスター繁殖システムは、管理者が自身のダンジョンをより強力、かつ個性的に構築するための核心的な仕組みです。捕獲した野生のモンスターを掛け合わせ、能力を引き継いだ新たな個体（卵）を生み出すことで、戦略的なダンジョン運営を可能にします。

## 2. 繁殖のプロセス

### 2.1 施設の利用
- 繁殖は、管理者エリア（拠点）に設置された「繁殖場（Breeding Center）」で行います。
- **繁殖コスト**: 繁殖には、親となる2体のモンスターと、以下のリソースが必要です。
    - **ゴールド**: `(親1のレベル + 親2のレベル) * 100` ゴールド
    - **魔力結晶 (`magic_crystal`)**: 5 個
- リソースが不足している場合、繁殖を行うことはできません。

### 2.2 卵の生成
- 繁殖を実行すると、親の特性を受け継いだ「モンスターの卵」が生成されます。
- 卵の状態では、将来生まれてくるモンスターの種類や一部の素質が伏せられている場合があります。

### 2.3 孵化
- **孵化時間**: 繁殖実行から孵化までには、個体の合計ステータスに比例した時間が必要です。
    - **計算式**: `孵化時間(分) = 合計ステータス / 10`
- **即時孵化**: 特定のアイテム（「孵化促進剤」等、検討中）を使用することで即座に孵化させることができます。
- 孵化したモンスターは「幼体」として所持モンスターリスト（[倉庫システム](Warehouse-System.md)）に加えられます。

## 3. 遺伝と継承

### 3.1 ステータスの継承
- 生まれてくるモンスターの基礎ステータス（HP、攻撃力、防御力等）は、両親のステータスの平均値をベースに算出されます。
- **継承計算式**:
    - `子の基礎値 = 両親の平均値 × (1.0 ± 0.05 〜 0.15)`
    - 親のステータスの平均に対し、5% 〜 15% のランダムな差異（振れ幅）が発生します。
- 高レベルの親同士を掛け合わせることで、より高い基礎能力を持つ個体が得られます。

### 3.2 特性の継承
- モンスターが持つ特殊なスキルや耐性、AIパターン（[戦闘システム](Combat-System.md) 参照）が、それぞれの特性ごとに **50%** の確率で継承されます。
- 特定の組み合わせでのみ発現する「合体技」や「特殊能力」が存在します。

### 3.3 突然変異 (Mutations)
- **確率**: 2%
- 低確率で、両親のどちらも持っていない特殊な色（色違い）や、極めて高い能力を持つ「変異種」が誕生することがあります。
- 特定のアイテム（**変異の薬**）を使用することで、この確率は **+10% (計12%)** に上昇します。

## 4. 戦略的活用

### 4.1 ダンジョン配置
- 繁殖によって強化されたモンスターを「マイ・ダンジョン」に配置することで、侵入者（プレイヤー）に対する防衛力を高めることができます。
- 階層の特性（火の階、水の階など）に合わせた耐性を持つモンスターを量産する、といった戦略が可能です。

### 4.2 モンスターの寿命と世代交代
- (検討中) モンスターには「繁殖回数」や「寿命」の設定があり、常に新しい世代への交代を促す仕組みを検討しています。
