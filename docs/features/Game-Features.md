# Game-Features

## 1. プロジェクト概要
RogueFは、Angular (v15) を使用して構築されたローグライクゲームのフロントエンドアプリケーションです。
バックエンドAPI (`/api`) と連携し、ダンジョンの探索、プレイヤーの状態管理、アイテムの取得などの機能を提供します。

## 2. ゲームメカニクス

### 2.1 プレイヤー属性
- **HP**: 体力。0になるとゲームオーバーが想定されます。
- **スタミナ**: 行動（移動等）によって消費されます。スタミナが切れると代わりにHPが減少します。
- **満腹度**: 探索における長期的なリソース。詳細は [満腹度システム](Hunger-System.md) を参照してください。
- **レベル・経験値**: プレイヤーの強さを表します。詳細は [経験値・レベルアップシステム](Leveling-System.md) を参照してください。
- **ステータス**: 攻撃力や防御力など、戦闘に影響する能力値です。
- **ゴールド**: ゲーム内通貨。

### 2.2 行動とスタミナ
- 移動やアイテム取得などの行動にはインターバル（`actionInterval`）が設定されており、連続した高速な行動は制限されます。
- `IntervalService` により、一定時間ごとにスタミナが自動回復します。

### 2.3 操作方法
- **移動**:
  - キーボード: `ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight`
  - Viライク: `h`, `j`, `k`, `l`
  - テンキー: `2`, `4`, `6`, `8`
- **アイテム取得**: `g`
- **階段の上り下り**: `<` (上), `>` (下)

### 2.4 マルチプレイヤーと権限
- **複数人プレイ**: 本ゲームは複数人での同時実行が可能です。
- **管理者権限**: バックエンドの管理者権限を持つユーザーは、ゲーム内の「管理者」として特別な操作が行えます。

### 2.5 管理者機能
- **世界構築**: 管理者は世界内にダンジョンを新規作成できます。
- **モンスター飼育・配置**: 自身で飼育したモンスターをダンジョン内に配置できます。
- **ショップ経営**: 武器などのアイテムを販売するアイテム屋の経営が可能です。
- **世界間移動の開放**: 他の世界の管理者と合意することで、互いの世界を行き来可能に設定できます。

## 3. 主要なゲームサイクル
1. **探索 (プレイヤー)**: ダンジョン内を移動し、マップを明らかにします。
2. **収集 (プレイヤー)**: 落ちているゴールドやアイテムを拾います。
3. **生存 (プレイヤー)**: スタミナを管理し、HPが尽きないように注意しながら深い階層を目指します。
4. **目標 (プレイヤー)**: 階段を見つけ、次の階層へと進みます。
5. **運営 (管理者)**: ダンジョンを作成し、モンスターやショップを配置して世界を拡張・管理します。
