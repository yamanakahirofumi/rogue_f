# 配布方法

本プロジェクトは Web アプリケーションとしてビルドされ、静的ファイルとして配信されます。

## 1. 配布形式
- **静的アセット**：`ng build` コマンドにより生成された `dist/` ディレクトリ内のファイルを、Web サーバー（Nginx, Apache, GitHub Pages 等）でホストします。
- **デプロイ手順**：
    1. `npm install` で依存関係をインストール。
    2. `npm run build` で本番用ビルドを実行。
    3. 生成された `dist/rogue-f` ディレクトリの内容をサーバーにアップロード。

## 2. メリット
- **クロスプラットフォーム**：モダンな Web ブラウザがあれば、OS を問わず実行可能です。
- **即時更新**：サーバー上のファイルを更新するだけで、ユーザーは常に最新バージョンを利用できます。
- **環境構築不要**：ユーザー側でのランタイムインストールや環境設定は不要です。

## 3. 関連ツール
- [Angular CLI](https://angular.io/cli)：ビルドおよび最適化ツール
- [GitHub Pages](https://pages.github.com/)：ホスティングサービスの一例
