# RogueF Documentation

このディレクトリには RogueF プロジェクトの仕様および開発に関するドキュメントが格納されています。

## ドキュメント一覧

- **[specification.md](specification.md)**:
  プロジェクトの全体像、システム構成、画面構成、ゲームメカニクス、APIリファレンス、データモデルなどの基本仕様を記述しています。
- **[development.md](development.md)**:
  現在の開発状況、既知のバグや不整合、テストの実行状況、今後の課題など、技術的・開発運用面に関する情報を記述しています。

## メンテナンス方針

- 新機能の追加や仕様の変更があった場合は、速やかに `specification.md` を更新してください。
- 開発上の課題やバグを発見した場合は、`development.md` に追記してください。
